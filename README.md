Este projeto, chamado Amigo Digital, foi desenvolvido utilizando HTML, CSS e JavaScript com o objetivo de criar um mascote virtual interativo.
A aplicação permite que o usuário interaja com o mascote, cuidando de sua energia e felicidade por meio de ações simples, como alimentar e brincar.
O mascote possui um nome, níveis de fome e felicidade e é representado visualmente por um emoji,
tornando a experiência mais amigável e intuitiva.

A estrutura do projeto é baseada na integração das três principais tecnologias da web. O HTML é responsável por organizar os elementos da página,
como o título, a área do mascote, as barras de status, os botões de interação e a mensagem exibida ao usuário. O CSS define o estilo visual da aplicação,
utilizando cores suaves, layout centralizado, bordas arredondadas e sombras, proporcionando uma interface agradável e de fácil compreensão. O JavaScript é responsável 
pela lógica do funcionamento do mascote, controlando seus estados e atualizando a interface de forma dinâmica.

No código JavaScript, foi criado um objeto que representa o mascote, contendo informações como nome, nível de fome, nível de felicidade e o emoji exibido na tela.
A aplicação utiliza funções para modificar esses valores conforme as ações do usuário e uma função principal para atualizar a interface, alterando as barras de progresso
e as mensagens exibidas. Por meio de estruturas condicionais, o sistema avalia o estado atual do mascote e exibe diferentes mensagens, indicando se ele está feliz, 
com pouca energia ou aguardando uma nova interação.

As interações acontecem através de botões que acionam eventos de clique. Ao alimentar o mascote, seu nível de energia aumenta, enquanto ao brincar, sua felicidade aumenta, 
porém com um pequeno gasto de energia, simulando uma situação mais realista. Após cada ação, a interface é atualizada automaticamente, refletindo as mudanças realizadas. 
Esse comportamento permite ao usuário compreender, de forma prática, como funciona a manipulação do DOM e a atualização dinâmica de elementos na tela.

O projeto Amigo Digital é ideal para iniciantes em programação, pois demonstra de maneira clara e prática conceitos fundamentais do desenvolvimento web, como uso de objetos,
funções, eventos e lógica condicional. Além de reforçar o aprendizado técnico, o projeto torna o estudo mais divertido e interativo, mostrando como a programação pode ser 
aplicada na criação de pequenas aplicações funcionais e visuais.
